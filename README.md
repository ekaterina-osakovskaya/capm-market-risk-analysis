# CAPM & Market Risk Analysis

## Overview

This project implements the Capital Asset Pricing Model (CAPM) using historical market data retrieved from Yahoo Finance.

The objective is to estimate systematic risk (beta), abnormal returns (alpha), and evaluate the relationship between an individual stock and the market through linear regression.

## Objectives

- Retrieve historical market data
- Calculate daily returns
- Estimate beta and alpha
- Perform linear regression
- Compare expected and realized returns
- Visualize the Security Characteristic Line

## Portfolio

Asset: Apple Inc. (AAPL)

Market Benchmark: SPDR S&P 500 ETF Trust (SPY)

## Metrics

- Daily Returns
- Beta
- Alpha
- Correlation
- R²
- Expected Return (CAPM)
